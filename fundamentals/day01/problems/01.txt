Question 1
A flowchart to display the greater of the two numbers.

Algorithm:-
1. input the first number(num1)
2. input the second number(num2)
3. if (num1>num2),then
4. display "num1 is greater than num2"
5. else if(num2>num1),then
6. display "num2 is greater than num1"
7. else
8. display "both numbers are equal"

Pseudo-code:-
Input num1
Input num2
if(num1>num2) then
    display "num1 is greater than num2"
else if(num2>num1) then
    display "num2 is greater than num1"
else
    display "both numbers are equal"
